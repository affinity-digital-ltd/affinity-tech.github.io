eBook Generator

What began as an in-house app to help us create eBooks has become a public facing tool to give back to the community what we have found a great timesaver. Currently, this is only capable of exporting to .ePub format which can be used in pass like iBooks.

Current features include the ability to reorder sections of the book by drag and dropping them, editing and deleting sections and through using Markdown to write content. Markdown is a very quick and easy way to style and control HTML. We have plans to introduce many more features such as .mobi export which can be used with Amazon's Kindle line of readers and support for adding front cover images and more control over the style of your content. Further down the road, we would even like to build integrated services to help have your book published or even printed. If you would like to see a certain feature integrated into the site, then we would love to hear from you. Ideas can be sent to us from our contact page form.

eBook Gem

Our eBook Gem is currently powering our eBook Generator site, details of which can be found above.

Installation

Add this line to your application's Gemfile:

gem 'ebook_generator'

And then execute:

$ bundle

Or install it yourself as:

$ gem install ebook_generator

Usage

Generate the tables needed to process the ebooks: rails generate ebook_generator

Migrate the database: rake db:migrate

Pass the id for the ebook you want to generate: EbookGenerator.generate_ebook(ebook.id)

This will then generate an ePub based on the values in the db and output to the /tmp folder.
